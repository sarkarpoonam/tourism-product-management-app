
# Tourism Product Management App

A responsive tourism dashboard to manage travel packages.

## Features
- Add travel packages
- Edit packages
- Delete packages
- Destination images
- Search destinations
- Country filter
- Responsive UI
- REST API integration using json-server

## Tech Stack
- HTML
- CSS
- JavaScript
- json-server

## Setup Instructions

1 Install NodeJS  
https://nodejs.org

2 Install json-server
npm install -g json-server

3 Start API
json-server --watch db.json

4 Open Website
Open index.html in your browser

API Endpoint:
http://localhost:3000/packages
